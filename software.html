Color Shapes

Color Shapes is a logic game and the main software developed by Nanoboot.

You need Java 8 to run Color Shapes 1.0.0 Color Shapes 2.0.0 is still in development and runs on Java 19

Screenshots:

More screenshots.

Color Shapes software:
  • Color Shapes Engine - core library for this logic game
  • Color Shapes Desktop - intended for PC (Windows, Linux, MacOS)
  • Color Shapes Server - intended for server
  • Color Shapes Client - library to connect to server
  • Color Shapes Web - intended for web browser
