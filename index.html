What is Nanoboot?

  1. Collection of free software mostly written in java : games, web apps, multi-purpose programming libraries
  2. Collection of topics, which we are interested in
  3. Collection of articles (encyclopedia)

History of Nanoboot

Development of Nanoboot software started in January 2016.
Nanoboot was publicly announced on 2nd December 2022.

Examples of software

Color Shapes (game)


Games:

  • Color Shapes Engine - core library for this logic game
  • Color Shapes Desktop - intended for PC (Windows, Linux, MacOS)
  • Color Shapes Server - intended for server
  • Color Shapes Client - library to connect to server
  • Color Shapes Web - intended for web browser
  • Simulife - screen-oriented game engine intended for web browser

Octagon

Octagon - specify, what you need to store, extend the functionality by existing plugins or develop a new one
