About

Nanoboot is collection of free software mostly written in Java programming language.

History

Software development started already in January 2016.
Nanoboot Project was publicly announced on 2nd December 2022.

Our mission

  1. To develop game Color Shapes "Color linez" clone
  2. To study game Color linez "Color Lines" clone
  3. To develop reimplementation of Color linez game using Java, JavaFX, ...
  4. To collect all webpages on the Internet related to game Color Lines and its clones
  5. To study game Color Lines and its clones
  6. To develop supportive libraries for game Color Shapes like Power Framework and DB Migration
  7. To study and learn technologies, which Color shapes uses. Java, JavaFX, SQLite, Git, Maven, ...
