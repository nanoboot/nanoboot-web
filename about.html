About

Nanoboot mission

  1. To develop game Color Shapes a "Color linez" clone
  2. To study game Color linez a "Color Lines" clone
  3. To reimplement game Color linez using Java, JavaFX, ...
  4. To list all clones of game Color Lines
  5. To collect Internet websites with content related to game Color Lines or its clones
  6. To develop other free software like Power Framework, JBugzilla, Octagon, DB Migration, ...
  7. To study and learn technologies, which Color Shapes uses. Java, JavaFX, SQLite, Git, Maven, ...
We are open to discuss new ideas and to extend are mission.

Nanoboot history

Software development started already in January 2016.
Nanoboot Project was publicly announced on 2nd December 2022.
